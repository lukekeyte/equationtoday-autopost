# equationtoday-autopost

Automated posting pipeline for [@equationtoday](https://www.instagram.com/equationtoday/) on Instagram. A GitHub Action runs daily to publish an equation carousel and a Reel via the Instagram Graph API.

This repo is public because GitHub Pages requires it to serve the image and video assets that the Instagram API fetches from. If you're looking for my research software, check out [PUFFIN](https://github.com/lukekeyte/PUFFIN) or [SIMBA](https://github.com/lukekeyte/SIMBA).
